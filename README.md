# 社群貼文洞察儀表板

門市社群貼文（FB／IG／Google 商家）的內容靈感與排程工具，單一 HTML 檔，離線可用。

## 功能
- **30 篇可直接複製的社群貼文**，依產品線（iPhone／iPad／Mac／Apple Watch／AirPods／配件／在地服務等）下拉收納
- **15 篇 IG 限動短版**文案
- 即時搜尋與篩選，快速找到對應主題的貼文
- 600 字實抓關鍵字索引，依產品線分組
- 內建出圖面板：每篇貼文一鍵「做成貼文圖」，自動配對 Phosphor 圖示，支援方形 1:1／橫式 4:3／商家 16:9 三種比例，PNG／JPG 下載
- 可 `Cmd + P` 列印成 A4

## 使用方式
直接用瀏覽器開啟 `index.html` 即可。出圖功能需連網（線上載入製圖工具與圖示資源）。

## 技術
- 純前端單檔 HTML／CSS／JavaScript，無後端、無建置流程
- 出圖使用 `html-to-image`，圖示使用 Phosphor Icons（皆由 CDN 載入）
